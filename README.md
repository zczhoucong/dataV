# show

    模板可以看作是特殊的项目页面，同样可以进行新增、修改、编辑、删除、发布等
    在创建页面选择模板时，应选择已发布的模板列表


### 需要新增组件列表
* 背景色块
* 当前时间显示(时间器)
* 天气预报
* 多行文本
* 轮播饼图


4大部分
* 组件本身 vue
* 组件模板 js
* 添加组件库 js
